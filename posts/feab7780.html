安全等于 <=>

在没使用NULL比较时 所有与NULL比较的运算结果都为NULL,比如 SELECT NULL = NULL; 结果为NULL而不是1

在使用安全等于以后可以正常使用NULL进行比较, SELECT NULL <=> NULL; 结果则为1
