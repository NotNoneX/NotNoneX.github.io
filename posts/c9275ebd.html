32_JavaScript_逻辑

JavaScript 布尔（逻辑）代表两个值之一：true 或 false。

布尔值

通常，在编程中，您会需要只能有两个值之一的数据类型，比如

  • YES / NO
  • ON / OFF
  • TRUE / FALSE

鉴于此，JavaScript 提供一种布尔数据类型。它只接受值 true 或 false。
