04_JavaScript_HTML_DOM_元素

    • JavaScript HTML DOM 元素

      本章向您讲解如何查找和访问 HTML 页面中的 HTML 元素。

      查找 HTML 元素

      通常，通过 JavaScript，您需要操作 HTML 元素。

      为了达成此目的，您需要首先找到这些元素。有好几种完成此任务的方法：

      • 通过 id 查找 HTML 元素
      • 通过标签名查找 HTML 元素
      • 通过类名查找 HTML 元素
      • 通过 CSS 选择器查找 HTML 元素
      • 通过 HTML 对象集合查找 HTML 元素
